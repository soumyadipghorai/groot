Tech Used

Python

Python is used as a mojor component of this project. All the data collection, preprocessing and model training are done in python. Libraries cv2, os, Numpy, pandas, pickle are used extensively

Object detection

For detecting objects in the image we have used yolo V5 along with custom implementation. For this we have considered around 85 classes. Robo flow is used to train on custom images.

Development

For development again python is used in the backend. Flask is used create the backend and for storing user data Firebase is used. In the frontend HTML, CSS, javascript and bootstrap is used.
